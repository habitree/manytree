# 바이럴 요소 분석

## 1. 왜 심리테스트를 공유하는가?

### 1.1 공유 심리학

```
┌─────────────────────────────────────────────────────┐
│                   자아 표현 욕구                      │
│  "이 결과가 나를 잘 설명해. 이게 나야!"              │
└─────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────┐
│                   사회적 연결 욕구                    │
│  "친구들과 비교해보고 싶어. 우리 유형 같을까?"       │
└─────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────┐
│                   대화 소재 욕구                      │
│  "재밌는 거 발견했어! 너도 해봐!"                    │
└─────────────────────────────────────────────────────┘
```

### 1.2 공유 동기 5가지

| 동기 | 설명 | 테스트 적용 |
|------|------|------------|
| **자아 표현** | 내가 어떤 사람인지 보여주고 싶음 | 매력적인 유형명 |
| **관계 형성** | 친구/연인과 공유하며 유대 강화 | 궁합 기능 |
| **정보 제공** | 유용한 정보를 공유하는 뿌듯함 | 실용적 조언 |
| **재미 공유** | 웃긴 거, 신기한 거 공유하고 싶음 | 유머 요소 |
| **대화 시작** | 뭔가 대화할 소재가 필요함 | 비교 가능한 결과 |

## 2. STEPPS 모델 적용

### 2.1 Social Currency (소셜 화폐)

> "공유하면 내가 멋져 보여야 한다"

**적용 전략:**
- **희소성**: "상위 5%만 나오는 결과" 느낌
- **특별함**: 독특하고 매력적인 유형명
- **우월감**: 긍정적이고 자랑할 만한 특성

```
❌ "당신은 평범한 유형입니다"
✅ "당신은 [숨겨진 천재성을 가진 창조자] 유형입니다"
```

### 2.2 Triggers (트리거)

> "일상에서 자연스럽게 떠올라야 한다"

**적용 전략:**
- **시즌 연결**: 계절/이벤트에 맞는 테스트
- **일상 연결**: 커피, 출근, 주말 등 일상 소재
- **트렌드 연결**: 유행하는 드라마, 밈, 키워드

```
1월: "새해 목표 유형 테스트"
12월: "크리스마스에 나는 어떤 유형?"
드라마 방영 시: "드라마 XX 캐릭터 중 나는?"
```

### 2.3 Emotion (감정)

> "감정이 움직여야 공유한다"

**적용 전략:**
- **공감**: "맞아, 이게 나야!" 느낌
- **놀라움**: "이걸 어떻게 알았지?" 느낌
- **웃음**: 유머 요소 삽입
- **감동**: 따뜻한 조언과 응원

**감정 유발 포인트:**
```
1. 질문에서: "이런 적 있지 않아?" 공감 유발
2. 결과에서: "당신의 이런 모습, 정말 특별해요" 감동
3. 조언에서: "가끔 힘들 수 있어요. 괜찮아요" 위로
```

### 2.4 Public (가시성)

> "눈에 띄어야 한다"

**적용 전략:**
- **결과 이미지**: 공유 시 눈에 띄는 디자인
- **유형 뱃지**: SNS 프로필에 올릴 수 있는 이미지
- **해시태그**: 검색/발견 가능한 태그

**결과 이미지 요소:**
```
┌─────────────────────────────┐
│  🌟 감성적인 힐러형 🌟      │
│                             │
│  "따뜻한 공감 능력의 소유자" │
│                             │
│  #ManyTree #심리테스트      │
│  manytree.kr               │
└─────────────────────────────┘
```

### 2.5 Practical Value (실용적 가치)

> "유용한 정보여야 공유한다"

**적용 전략:**
- **자기 이해**: 내가 왜 그런지 알게 됨
- **관계 개선**: 연인/친구 이해에 도움
- **실천 가능한 팁**: 바로 적용할 수 있는 조언

```
✅ "이런 상황에서 이렇게 해보세요"
✅ "당신의 강점을 이렇게 활용해보세요"
✅ "파트너가 이 유형이라면 이렇게 대해주세요"
```

### 2.6 Stories (스토리)

> "이야기로 전달되어야 한다"

**적용 전략:**
- **유형에 서사 부여**: 영웅의 여정처럼 표현
- **성장 스토리**: 현재 → 미래 가능성
- **관계 스토리**: 다른 유형과의 상호작용

```
"당신은 [고독한 전략가] 유형이에요.

조용히 혼자서 생각하는 시간이 필요한 당신.
사람들은 당신이 무심해 보인다고 생각하지만,
사실 누구보다 깊이 생각하고 있죠.

언젠가 당신의 전략이 빛을 발할 때,
모두가 당신의 혜안에 놀랄 거예요."
```

## 4. 플랫폼별 최적화

### 4.1 카카오톡

**특성:**
- 1:1 또는 그룹 채팅 공유
- 썸네일 이미지 중요
- "너도 해봐" 유도

**최적화:**
```
- OG 이미지: 1200x630px
- 제목: 30자 이내
- 설명: 60자 이내
- 행동 유도: "나는 OO 유형! 너도 해봐!"
```

### 4.2 인스타그램 스토리

**특성:**
- 세로형 이미지 (9:16)
- 24시간 후 사라짐
- 스와이프 업 링크

**최적화:**
```
- 결과 이미지: 1080x1920px
- 유형 뱃지 디자인
- 스토리용 템플릿 제공
```

### 4.3 트위터(X)

**특성:**
- 짧은 텍스트 + 이미지
- 리트윗으로 확산
- 해시태그 검색

**최적화:**
```
- 결과 요약: 140자 이내
- 해시태그: #심리테스트 #MBTI #OO유형
- 결과 이미지: 1200x675px
```

## 5. 바이럴 성공 사례 분석

### 사례 1: MBTI 테스트
**성공 요인:**
- 16가지 유형의 희소성
- "나는 INFJ야" 정체성 표현
- 궁합/케미 논쟁 유발

### 사례 2: 연애 캐릭터 테스트
**성공 요인:**
- 귀여운 캐릭터 이미지
- 연인과 함께 하는 재미
- 결과 밈화 가능

### 사례 3: 동물 성격 테스트
**성공 요인:**
- 친숙한 동물 비유
- 프로필 사진으로 활용
- "나는 고양이야" 자기 표현

## 6. 측정 지표

### 핵심 KPI

| 지표 | 설명 | 목표 |
|------|------|------|
| 공유율 | 테스트 완료자 중 공유 비율 | 30%+ |
| 바이럴 계수 | 1명이 가져오는 추가 사용자 | 1.5+ |
| 소셜 클릭률 | 공유된 링크 클릭률 | 10%+ |
| 재방문율 | 다른 테스트 하러 다시 방문 | 40%+ |

### 측정 방법
```
1. UTM 파라미터로 공유 채널 추적
2. 공유 버튼 클릭 이벤트 추적
3. 리퍼러 분석으로 유입 경로 파악
4. 코호트 분석으로 재방문 패턴 확인
```
