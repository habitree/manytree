```

│        자아실현 욕구                 │  ← 잠재력, 성장, 목표
│     (Self-Actualization)            │
├─────────────────────────────────────┤
│         존중 욕구                    │  ← 인정, 성취, 자존감

│         안전 욕구                    │  ← 안정, 건강, 재정
```

## 3. 인간의 근본 욕구 기반

### 3.1 핵심 욕구 8가지

```
1. 정체성 확인 욕구   → "나는 어떤 사람인가?"
2. 관계 욕구         → "나는 잘 사랑받고 있는가?"
3. 인정 욕구         → "나는 가치 있는 사람인가?"
4. 성장 욕구         → "나는 발전하고 있는가?"
5. 안정 욕구         → "나는 안전한가?"
6. 자유 욕구         → "나는 자유로운가?"
7. 쾌락 욕구         → "나는 즐겁게 살고 있는가?"
8. 의미 욕구         → "내 삶에 의미가 있는가?"
```

### 3.2 욕구별 테스트 매핑

| 욕구 | 핵심 질문 | 테스트 주제 |
|------|----------|-------------|
| 정체성 | 나는 누구? | MBTI, 성격유형, 전생, 동물 자아 |
| 관계 | 나는 사랑받는가? | 연애유형, 애착유형, 사랑의 언어 |
| 인정 | 나는 잘하고 있나? | 능력 테스트, 재능 발견, IQ |
| 성장 | 어떻게 발전할까? | 잠재력, 미래 자아, 성장 영역 |
| 안정 | 괜찮은 건가? | 멘탈 건강, 스트레스, 불안 |
| 자유 | 진정한 나인가? | 진짜 성격, 숨겨진 자아, 억압 |
| 쾌락 | 뭘 하면 즐거울까? | 취미, 여행, 음식, 놀이 스타일 |
| 의미 | 왜 사는가? | 가치관, 인생관, 철학 유형 |

## 4. 주제 선정 평가 매트릭스

### 평가 기준 (각 1-5점)

| 기준 | 설명 | 가중치 |
|------|------|--------|
| 욕구 강도 | 얼마나 강한 욕구를 자극하는가 | 25% |
| 공유 가능성 | SNS에서 공유하고 싶은가 | 25% |
| 검색량 | 관련 키워드 검색량 | 20% |
| 차별성 | 기존 테스트와 얼마나 다른가 | 15% |
| 제작 용이성 | 질문/결과 설계가 쉬운가 | 15% |

### 평가 공식

```
점수 = (욕구강도 × 0.25) + (공유가능성 × 0.25) + (검색량 × 0.2) + (차별성 × 0.15) + (제작용이성 × 0.15)
```

우선순위: **4.0점 이상** → 즉시 제작 | **3.0~3.9점** → 검토 후 제작 | **3.0 미만** → 보류
