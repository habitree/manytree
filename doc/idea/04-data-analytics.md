## 조사 배경

### 왜 데이터 분석인가?

- **자기 인식**: 독서 습관을 객관적으로 파악
- **동기 부여**: 진행 상황 시각화로 성취감
- **개선 기회**: 데이터 기반 목표 설정
- **공유 가치**: SNS 공유 콘텐츠로 바이럴

### 데이터 분석의 가치

1. **회고**: 지난 독서 여정 돌아보기
2. **현황**: 현재 상태 파악
3. **예측**: 미래 독서 계획 수립
4. **비교**: 목표 대비 진척도

## 기능 아이디어

### 1. 독서 캘린더 (Reading Calendar)

#### GitHub 스타일 히트맵
```
     1월    2월    3월    4월    5월
월   ⬜⬜⬜🟩⬜🟩🟩⬜⬜🟩🟩🟩...
화   ⬜🟩🟩🟩⬜🟩🟩🟩⬜🟩🟩🟩...
수   🟩🟩🟩🟩⬜🟩🟩🟩🟩🟩🟩🟩...
목   🟩🟩⬜🟩🟩🟩🟩🟩🟩🟩⬜🟩...
금   ⬜🟩🟩🟩🟩🟩⬜🟩🟩🟩🟩🟩...
토   🟩🟩🟩⬜🟩🟩🟩🟩⬜🟩🟩🟩...
일   🟩⬜🟩🟩🟩⬜🟩🟩🟩🟩🟩⬜...

🟩 독서 완료  ⬜ 미독서  🟢 높은 활동량
```

#### 색상 레벨
| 레벨 | 조건 | 색상 |
|------|------|------|
| 0 | 독서 안 함 | 회색 |
| 1 | 10분 미만 | 연한 초록 |
| 2 | 10-30분 | 중간 초록 |
| 3 | 30분-1시간 | 진한 초록 |
| 4 | 1시간 이상 | 매우 진한 초록 |

#### 상호작용
- 날짜 클릭 시 해당 일 독서 상세
- 월/주/일 뷰 전환
- 스트릭 하이라이트

### 2. Year in Books (연간 독서 리포트)

#### 슬라이드 구성
```
Slide 1: 인트로
"당신의 2026년 독서 여정"

Slide 2: 총 독서량
📚 42권 완독
📖 12,456 페이지
⏱️ 156시간

Slide 3: 최고의 책
⭐ 가장 높이 평가한 책
💕 가장 오래 읽은 책
⚡ 가장 빨리 읽은 책

Slide 4: 장르 분석
[파이 차트]
소설 45%
자기계발 25%
역사 15%
기타 15%

Slide 5: 독서 패턴
🌙 밤 독서가 (주로 밤에 기록)
📅 주말 독서가 (주말 활동량 높음)
🔥 7월이 가장 열정적!

Slide 6: 나무 성장
[나무 이미지]
씨앗 → 성목으로 성장!

Slide 7: 스트릭 기록
최장 스트릭: 32일
총 독서일: 245일

Slide 8: 2027년 목표
"내년에는 50권 도전?"
```

#### 공유 최적화
- 인스타그램 스토리 비율 (9:16)
- 카카오톡 썸네일 최적화
- 트위터 카드 형식

### 3. 독서 통계 대시보드

#### 기본 통계
```
┌─────────────────────────────────┐
│  📊 나의 독서 통계              │
├─────────────────────────────────┤
│  이번 달                        │
│  ├── 읽은 책: 4권              │
│  ├── 읽은 페이지: 1,234p       │
│  ├── 독서 시간: 18시간          │
│  └── 평균 평점: 4.2점          │
│                                 │
│  올해 누적                      │
│  ├── 총 완독: 28권             │
│  ├── 목표 달성률: 56%          │
│  └── 작년 대비: +12%           │
└─────────────────────────────────┘
```

#### 차트 종류
| 차트 | 데이터 | 목적 |
|------|--------|------|
| 라인 차트 | 월별 독서량 | 트렌드 파악 |
| 바 차트 | 장르별 분포 | 취향 분석 |
| 도넛 차트 | 완독/진행/미독 | 상태 파악 |
| 레이더 차트 | 장르 다양성 | 균형 확인 |

### 4. 독서 목표 트래커

#### 목표 유형
```
연간 목표
├── 권수 목표: 50권/년
├── 페이지 목표: 15,000p/년
└── 시간 목표: 300시간/년

월간 목표
├── 권수: 4권/월
└── 새 장르 도전: 1권/월

주간 목표
├── 독서일: 5일/주
└── 독서 시간: 5시간/주
```

#### 진행률 시각화
```
2026년 독서 목표: 50권

[████████████████░░░░░░░░░░░░░░] 56%
                28/50권 완료

📅 남은 기간: 168일
📚 필요 속도: 0.13권/일 (약 7일에 1권)
💡 현재 속도 유지 시 예상: 47권 (목표 미달)
```

### 5. 독서 인사이트

#### 패턴 분석
| 인사이트 | 예시 |
|----------|------|
| 시간대 | "주로 밤 10-11시에 독서해요" |
| 요일 | "일요일 독서량이 가장 많아요" |
| 계절 | "겨울에 독서량이 증가해요" |
| 장르 주기 | "소설 → 자기계발 패턴" |

#### 비교 분석
| 비교 대상 | 내용 |
|----------|------|
| 작년 대비 | 독서량, 장르 변화 |
| 목표 대비 | 달성률, 예상 완료일 |
| 평균 대비 | 전체 사용자 평균과 비교 |

#### 추천 인사이트
```
💡 인사이트

"최근 자기계발서를 많이 읽었어요.
기분 전환으로 소설은 어떨까요?"

"7월에 독서량이 급격히 줄었어요.
휴가 시즌이었나요? 8월엔 다시 화이팅!"

"최장 스트릭 32일! 역대 최고 기록이에요.
이번엔 50일에 도전해보세요!"
```

### 6. 기분/감정 태그

#### 독서 시 기분 기록
```
오늘 독서하면서 느낀 기분은?

[😊 즐거움] [🤔 생각에 잠김] [😢 슬픔]
[😮 놀라움] [😌 평온함] [🔥 열정적]
```

#### 기분 분석
- 책 장르와 기분 연관성
- 기분별 추천 책
- 월별 기분 변화

### 7. 독서 마일스톤

#### 마일스톤 카드
```
🎉 축하합니다!

┌─────────────────────────┐
│  📚 100권 클럽 달성!     │
│                         │
│  2024.01 ~ 2026.01     │
│  총 2년 걸렸어요        │
│                         │
│  [공유하기] [저장하기]   │
└─────────────────────────┘
```

| 10권 클럽 | 10권 완독 | 배지 + 나무 성장 |
| 50권 클럽 | 50권 완독 | 프리미엄 배지 |
| 100권 클럽 | 100권 완독 | 레전드 배지 |
| 10,000 페이지 | 누적 페이지 | 숨겨진 배지 |
| 365일 스트릭 | 1년 연속 | 최고 등급 배지 |

## ReadingTree 적용 방안

### Phase 1: 기본 통계 (1-2개월)

1. **독서 캘린더**
   - GitHub 스타일 히트맵
   - 스트릭 표시

2. **기본 대시보드**
   - 월간/연간 통계
   - 진행 중인 책 목록

### Phase 2: 목표 & 인사이트 (2-4개월)

1. **목표 트래커**
   - 연간/월간 목표 설정
   - 진행률 시각화

2. **기본 인사이트**
   - 패턴 분석
   - 장르 분포

### Phase 3: Year in Books (4-6개월)

1. **연간 리포트**
   - 인터랙티브 슬라이드
   - SNS 공유 최적화

2. **고급 인사이트**
   - 기분 태그
   - 예측 분석

## 예상 효과 및 리스크

### 예상 효과

| 지표 | 예상 변화 |
|------|----------|
| 앱 재방문 | +25% (통계 확인) |
| 공유 횟수 | 월 5,000회 (Year in Books) |
| 목표 설정률 | 활성 유저의 60% |
| 리텐션 | +15% |

### 잠재적 리스크

| 리스크 | 영향도 | 대응 방안 |
|--------|--------|----------|
| 부정적 통계 | 낮음 | 긍정적 프레이밍 |
| 비교 스트레스 | 낮음 | 개인 성장 강조 |
| 데이터 부족 | 중간 | 최소 데이터로도 의미 있는 표시 |

### 성공 지표

- Year in Books 생성률: 활성 유저의 70%
- Year in Books 공유율: 생성 유저의 40%
- 목표 설정 유저: 전체의 50%
- 주간 대시보드 조회: DAU의 30%
