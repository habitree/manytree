## 조사 배경

### 왜 게이미피케이션인가?

- **습관 형성의 어려움**: 독서는 장기적 노력이 필요하지만 즉각적 보상이 부족
- **동기 부여 필요**: 외재적 동기(포인트, 배지)가 내재적 동기로 전환될 수 있음
- **검증된 효과**: Duolingo, Habitica 등에서 리텐션 향상 입증

### 주요 참고 자료

- "Hooked: How to Build Habit-Forming Products" - Nir Eyal
- "Gamification by Design" - Gabe Zichermann
- Octalysis Framework - Yu-kai Chou

## 벤치마킹 분석

### 1. Duolingo

| 요소 | 설명 | 효과 |
|------|------|------|
| 스트릭 | 연속 학습일 표시 | Day 7 리텐션 +40% |
| XP 시스템 | 경험치 기반 레벨업 | 학습량 증가 |
| 리그 시스템 | 주간 리더보드 | 경쟁심 유발 |
| 하트 시스템 | 실수 제한 | 집중력 향상 |
| 스트릭 프리즈 | 하루 쉬어도 스트릭 유지 | 이탈 방지 |

**핵심 인사이트**: 손실 회피(Loss Aversion)를 활용한 스트릭이 가장 효과적

### 2. Forest

| 요소 | 설명 | 효과 |
|------|------|------|
| 나무 심기 | 집중 시간에 나무 성장 | 시각적 보상 |
| 숲 만들기 | 누적 성과 시각화 | 장기 동기 |
| 실제 나무 심기 | 가상 코인으로 기부 | 사회적 의미 |

**핵심 인사이트**: 시각적 성장 메타포가 강력한 동기 부여

### 3. Habitica

| 요소 | 설명 | 효과 |
|------|------|------|
| RPG 캐릭터 | 습관이 캐릭터에 영향 | 책임감 |
| 파티 시스템 | 함께 퀘스트 수행 | 사회적 책임 |
| 보스 레이드 | 팀 목표 달성 | 협동 동기 |

**핵심 인사이트**: 사회적 책임감이 개인 목표보다 강력

### 4. Nike Run Club

| 요소 | 설명 | 효과 |
|------|------|------|
| 마일스톤 배지 | 거리별 배지 | 성취감 |
| 도전 과제 | 월간/시즌 챌린지 | 지속적 참여 |
| 음성 코칭 | 실시간 격려 | 동기 부여 |

## 기능 아이디어

### 1. 스트릭 시스템 (Streak System)

#### 기본 구조
```
연속 독서일수 추적
├── 매일 독서 기록 시 스트릭 +1
├── 하루 놓치면 스트릭 리셋
├── 스트릭 프리즈 (월 2회 무료)
└── 스트릭 복구 (유료 or 친구 도움)
```

#### 마일스톤 보상
| 스트릭 | 보상 | 나무 테마 연계 |
|--------|------|---------------|
| 7일 | 불꽃 이모지 | 새싹 뱃지 |
| 30일 | 실버 불꽃 | 가지 뱃지 |
| 100일 | 골드 불꽃 | 열매 뱃지 |
| 365일 | 다이아몬드 불꽃 | 숲 뱃지 |

### 2. 배지 시스템 (Badge System)

#### 배지 카테고리
```
독서량 배지
├── 첫 책 완독 (First Leaf)
├── 10권 클럽 (Branch Grower)
├── 50권 클럽 (Fruit Bearer)
├── 100권 클럽 (Forest Keeper)
└── 1000권 클럽 (Legendary Tree)

장르 배지
├── 소설 마스터
├── 자기계발 전문가
├── 역사 탐험가
└── 과학 덕후

특별 배지
├── 새벽 독서가 (오전 5시 이전 기록)
├── 올빼미 독서가 (자정 이후 기록)
├── 속독왕 (일주일에 3권)
├── 정독왕 (한 권에 1달)
└── 재독자 (같은 책 2번 이상)
```

### 3. 레벨 시스템 (Level System)

#### XP 획득 구조
| 활동 | XP |
|------|-----|
| 책 기록 | +10 |
| 책 완독 | +50 |
| 리뷰 작성 | +30 |
| 메모 작성 | +5 |
| 스트릭 유지 | +5/일 |
| 친구 추천 | +100 |

#### 레벨 구간
```
Lv 1-10: 씨앗 (Seed)
Lv 11-20: 새싹 (Sprout)
Lv 21-30: 묘목 (Sapling)
Lv 31-50: 어린 나무 (Young Tree)
Lv 51-75: 성목 (Mature Tree)
Lv 76-100: 고목 (Ancient Tree)
```

### 4. 독서 퀘스트 (Reading Quests)

#### 일일 퀘스트
- 오늘 10분 이상 독서하기
- 책에 메모 1개 남기기
- 새 책 시작하기

#### 주간 퀘스트
- 7일 연속 독서하기
- 책 1권 완독하기
- 리뷰 1개 작성하기

#### 월간 챌린지
- 새로운 장르 도전하기
- 500페이지 이상 책 완독
- 친구 1명 초대하기

### 5. 나무 성장 시각화

#### 성장 단계
```
Stage 1: 씨앗 (0-5권)
    └── 작은 씨앗 이미지

Stage 2: 새싹 (6-10권)
    └── 땅에서 싹이 나옴

Stage 3: 묘목 (11-25권)
    └── 작은 나무 형태

Stage 4: 어린 나무 (26-50권)
    └── 잎이 달린 나무

Stage 5: 성목 (51-100권)
    └── 크고 무성한 나무

Stage 6: 고목 (101권+)
    └── 장엄한 거대 나무
```

#### 꾸미기 요소
- 잎사귀 색상 (계절/테마)
- 나무 주변 장식 (동물, 꽃)
- 배경 변경 (숲, 공원, 정원)
- 특별 이펙트 (반짝임, 바람)

### 6. 리더보드 (Leaderboard)

#### 리더보드 종류
- 친구 랭킹 (1주일 기준)
- 전체 랭킹 (월간)
- 장르별 랭킹

#### 랭킹 기준
- 완독 권수
- XP 획득량
- 스트릭 일수
- 리뷰 작성 수

## ReadingTree 적용 방안

### Phase 1: 핵심 기능 (1-2개월)

1. **스트릭 시스템**
   - 연속 독서일 추적
   - 스트릭 프리즈 기능
   - 마일스톤 알림

2. **기본 배지**
   - 10가지 핵심 배지
   - 배지 컬렉션 페이지

### Phase 2: 성장 시각화 (2-3개월)

1. **나무 성장**
   - 6단계 성장 시스템
   - 기본 꾸미기 요소

2. **레벨 시스템**
   - XP 획득 로직
   - 레벨업 축하 모달

### Phase 3: 소셜 & 퀘스트 (3-4개월)

1. **리더보드**
   - 친구 랭킹
   - 주간/월간 랭킹

2. **퀘스트 시스템**
   - 일일/주간 퀘스트
   - 월간 챌린지

## 예상 효과 및 리스크

### 예상 효과

| 지표 | 예상 변화 |
|------|----------|
| Day 7 리텐션 | +15~25% |
| DAU | +20~30% |
| 평균 세션 시간 | +2분 |
| 월간 완독 수 | +30% |
| 앱 스토어 평점 | +0.3점 |

### 잠재적 리스크

| 리스크 | 영향도 | 대응 방안 |
|--------|--------|----------|
| 외재적 동기 과의존 | 중간 | 내재적 보상 강조 |
| 스트릭 스트레스 | 중간 | 스트릭 프리즈, 격려 메시지 |
| 과잉 게이미피케이션 | 낮음 | 점진적 도입, A/B 테스트 |
| 치팅 시도 | 낮음 | 합리적 검증 로직 |

### 성공 지표

- 7일 이상 스트릭 유저: 전체의 30% 이상
- 배지 3개 이상 획득: 전체의 50% 이상
- 나무 스크린샷 공유: 월 1,000회 이상
